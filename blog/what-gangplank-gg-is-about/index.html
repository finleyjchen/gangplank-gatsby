what gangplank.gg is about

December 4th, 2018

Introduction

Welcome to gangplank.gg! This is a side project created by your friendly GP main inlorfaze.

I created this website based off a reddit post idea, you can see it here.

The website's purpose is to provide information about Gangplank that other site's don't - op.gg, champion.gg, and other websites provide plenty data for you to analyze matches, however this site aggregates vital information for you from other experienced Gangplank players.

It's a one-of-a-kind one trick heaven!

Organization

I'm organizing this website into posts - these posts will have several categories, from talking about meta builds and techniques to guides and matchups.
